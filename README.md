# Silhouette Master Pro

## 公開サイト

- メイン画面  
  https://soundcruiser.github.io/silhouette-quiz/

- リモコン画面  
  https://soundcruiser.github.io/silhouette-quiz/remote.html

  
クイズイベント向けのシルエットクイズ進行ツールです。  
画像をシルエット表示しながらスライドアニメで徐々に明かしていく、イベント演出に特化した Web アプリケーションです。

---

## 機能概要

- **CONFIG モード** — セット（カテゴリ）・問題の登録・並び替え・設定
- **PLAY モード** — オープニング → カテゴリ → クイズ → エンディングの一連の進行
- **スマートフォンリモコン** (`remote.html`) — PeerJS 経由で PC と接続し、手元から操作
- **設定の保存／読込** — JSON 形式でイベントごとに設定をエクスポート・インポート
- **カスタムサウンド** — 各演出シーンに独自の SE / BGM を割り当て可能
- **PWA 対応** — スタンドアロンアプリとしてインストール可能

### ローカルで ES Modules を試すとき

`index.html` は `type="module"` で `./js/main.js` を読み込みます。`file://` 直開きでは環境によってモジュールがブロックされることがあるため、`npx serve .` や VS Code の Live Server など**静的ファイル用のローカル HTTP サーバー**で開いてください。

## セットアップ

ローカルサーバを立ち上げてブラウザで `index.html` を開くだけで動作します（File System Access API を使用するため、ローカルファイルでは一部機能が制限される場合があります）。

```bash
# 例: Python の簡易サーバ
python -m http.server 8080
# ブラウザで http://localhost:8080 を開く
```

## 使い方

### 1. フォルダ接続

ナビの **📁 フォルダ接続** をクリックし、画像が格納されたルートフォルダを選択します。  
フォルダ直下にカテゴリ用サブフォルダを作成し、各フォルダ内に問題画像（JPEG / PNG / GIF など）を入れておきます。

```
my-event/
├── silhouette-quiz-config.json   ← 保存時に自動生成
├── _sounds/                      ← 保存時に自動生成（カスタム音声）
├── 動物/
│   ├── 01_ライオン.jpg
│   └── 02_ペンギン.jpg
└── 食べ物/
    ├── 01_ラーメン.jpg
    └── 02_寿司.jpg
```

### 2. CONFIG モードで設定

| 項目 | 内容 |
|------|------|
| **速度 (s)** | 爆速 / 中速 / 低速それぞれのスライド時間（秒） |
| **カウントダウン** | スライド開始前のカウントダウン秒数（ON/OFF 切替可） |
| **スライド後の考え中** | スライド終了後に表示するオーバーレイのテキスト |
| **浮遊シェイプ** | オープニング・カテゴリ・エンディング・考え中に表示する装飾文字（8点） |
| **タイトル** | オープニングに表示するメインタイトル・サブタイトル |

カテゴリカードからは、各問題ごとに **シルエット / カラー** の切り替えが可能です。  
**すべてカラー / すべてシルエット** ボタンで一括設定もできます。

カテゴリには **スライドモード**（徐々に明かす）と **じっくりモード**（静止画を表示）の 2 種類があります。

### 3. 設定の保存・読込

- **💾 保存** — JSON と音声ファイルをフォルダへ書き出し
- **📂 読込** — 保存済みフォルダを選択して一括読込（`silhouette-quiz-config.json` を自動検出）

### 4. PLAY モードで進行

ショーは以下の順序で進行します。

```
オープニング → カテゴリ → クイズ（問題） → エンディング
```

**ショー画面（オープニング / カテゴリ / エンディング）**

| 操作 | 方法 |
|------|------|
| 次へ進む | `Space` / `Enter` / 画面の「NEXT ▶」ボタン |

**クイズ画面**

| 操作 | キー | ボタン |
|------|------|--------|
| スライド開始（爆速） | `1` | 爆速 |
| スライド開始（中速） | `2` | 中速 |
| スライド開始（低速） | `3` | 低速 |
| 正解を表示 | `Space` | ★ 正解 |
| 一時停止 / 再開 | `P` | ⏸ |
| リプレイ | `R` | ↻ |
| 前の問題 | `←` | ◀ |
| 次の問題 | `→` | ▶ |
| フルスクリーン | `F` | — |
| ヘルプ表示 | `H` / `?` | — |
| スキップ / キャンセル | `Esc` | — |

> **注意**: カウントダウン中・スライド再生中は `←` `→` `1` `2` `3` は無効です。  
> じっくりモードでは `1` `2` `3` `P` `R` は無効（正解と前後移動のみ）。

## スマートフォンリモコン

PC と同じネットワーク上でスマートフォンから `remote.html` を開くと、PeerJS（WebRTC）経由でリモコンとして操作できます。

1. PLAY モード起動後、画面に表示された **接続 ID（`sq-xxxxxx`）** を確認
2. スマートフォンで `remote.html` を開き、ID を入力して **接続**
3. QR コードをスキャンして接続することも可能

接続後は、スライドの速度切替・正解表示・前後移動・一時停止をスマートフォンから操作できます。

---

## カスタムサウンド

CONFIG モードのサウンドセクションで、以下のシーンに独自の音声ファイルを設定できます。

| スロット | タイミング |
|----------|-----------|
| BGM | 全体のバックグラウンドミュージック |
| 爆速 / 中速 / 低速 | 各速度でスライド開始時 |
| 正解 | 正解表示時 |
| カウントダウン | カウントダウン中 |
| オープニング | オープニング画面 |
| カテゴリ | カテゴリ表示時 |
| 問題イントロ / アフター | 問題前後 |
| エンディング | エンディング画面 |
| 考え中ループ | スライド終了後の「考え中」オーバーレイ中（BGM を自動でダックします） |

音声を設定しない場合は、内蔵のデフォルトサウンドが再生されます。
